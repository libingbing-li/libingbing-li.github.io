Flex

Flex布局是弹性布局(Flexible Box)，为盒模型提供最大的灵活性。
任何一个容器或行内元素都可以指定为flex布局。
设为flex布局后，子元素的float、clear、vertical-align属性失效。
采用flex布局的元素，称为flex容器(flex container)，简称容器。
容器的所有子元素自动成为容器成员，称为flex项目(flex item)，简称项目。

容器默认存在两根轴：
  • 水平的主轴 (main axis)
  • 主轴的开始位置 (main start) 是与边框的交叉点
  • 主轴的结束位置 (main end)
  • 垂直的交叉轴 (cross axis)
  • 交叉轴的开始位置 (cross start)
  • 交叉轴的结束位置 (cross end)

项目默认沿主轴排列，单个项目占据空间称为：
  • 主轴空间 (main size)
  • 交叉轴空间 (cross size)

Flex语法

指定为Flex布局
块元素 .box{ display: flex; }
行内元素 .box{ display: inline-flex; }
Webkit内核的浏览器 .box{ display: -webkit-flex; display: flex; /*保证兼容性*/ }

定义项目在主轴上的对齐方式
【justify:整理版面 content:内容】 		flex-start(默认) 左对齐 假设主轴从左到右

cneter 居中

align-items
定义项目在交叉轴上的对齐方式
【align:排列】 		stretch(默认)
【stretch:伸展，张开】		 若项目未设置高度或设为auto，
将沾满整个容器的高度		 假设交叉轴从上到下
flex-start 交叉轴的起点对齐
flex-end 交叉轴的终点对齐
center(默认 ) 交叉轴的中点对齐
baseline 项目的第一行文字的基线对齐
align-content
定义了多根轴线(多行)的对齐方式
如果项目只有一根轴线，则该属性不起作用 		stretch(默认) 轴线占满整个交叉轴 假设交叉轴从上到下
flex-start 与交叉轴的起点对齐
flex-end 与交叉轴的终点对齐
center 与交叉轴的中点对齐

所以，轴线之间的间隔比轴线与边框的间隔大一倍


项目属性
order
此属性定义项目的排列顺序 		数值 (默认0) 数值越小，排列越靠前
flex-grow
此属性定义项目的放大比例 		数值 (默认0) 为0时，即使存在剩余空间，也不放大
若所有项目的flex-grow都为1，则等分剩余空间(若有)
若其中一个的flex-grow为2，则其占据的剩余空间将比其他项目多一倍
flex-shrink
此属性定义项目的缩小比例
【shrink:缩水】 		数值 (默认1) 为1时，如果空间不足，该项目缩小
负值对该属性无效
若所有项目的flex-shrink为1，则空间不足时将等比例缩小
若有一为0，则其不缩小[示例中项目2此属性为0]
flex-basis
此属性定义了在分配多余空间之前，项目占据的主轴空间(main size)
浏览器根据这个属性，计算主轴是否有多余空间
【basis:基础】 		length:[如350px] (默认auto) auto:项目的本来大小
可设置成和width，height属性一样的值		 无示例
flex 是flex-grow(必选)，flex-shrink,flex-basis的简写(默认:0 1 auto)
有两个快捷值:auto(1 1 auto),none(0 0 auto)
建议优先使用此属性，因为浏览器会推算相关值
align-self
此属性允许单个项目有与其他项目不一样的对齐方式，可覆盖
【self:自身】 		同align-items属性一致 (默认auto) auto表示继承父元素的align-items属性，若无父元素，则等同stretch

双项目

display: flex;

}

(主轴方向默认)设置项目在主轴上的对齐方式(box属性:justify-content)为space-between，使之水平两端对齐，项目之间间隔相等(与边框无间隔)

display: flex;
flex-direction: column;
justify-content: space-between;
}

设置主轴方向(flex-direction)为 column(垂直上至下),再设置项目在主轴上的对齐方式(box属性:justify-content)为space-between，使之垂直两端对齐，项目之间间隔相等

flex-direction: column;
justify-content: space-between;
align-items: center;

display: flex;
}

align-self: center;
}

justify-content: space-between;
}

align-self: flex-end;
}

首先设置水平两端对齐(justify-content: space-between),再设置item2单独交叉轴对齐方式(align-self)为 flex-end，造成此种效果。

flex的值:auto(1 1 auto),none(0 0 auto)

圣杯布局

三栏布局是最常见的一种网站布局，又分为圣杯布局(Holy Grail Layout)和双飞翼布局。。
页面从上到下，分为头部(header)、躯干(body)、尾部(footer)。
其中躯干水平分为三栏(如果页面太小则垂直展现),从左到右为:导航、主栏、副栏。
圣杯布局就是两侧宽度固定，主栏宽度auto。
双飞翼布局就是主栏宽度固定，两侧宽度相等且变换。

【CSS分析(正常屏幕)】
在正式的圣杯布局中，box的min-height应为100vh(占满全屏)，本次为了演示只作30vh
将box设为flex，改变主轴方向(flex-direction: column;)，使header、body、footer三块垂直顺序排下，之后设置header、footer的flex为1，bodywei4，以自动分配高度，使三者按比例占据页面
之后设置body为flex，使其中main、nav、aside成为项目。此时设置main自动扩张(Flex:1),nav和aside不扩张，但有固定的占据主轴空间(flex: 0 0 12em[em:一个相对的度量单位，是一个字符的高度font-size])
设置nav的order为-1，使order成为主轴上第一个项目(放在最左边)。
【CSS分析(小于等于400px屏幕)】
此时使用@media (max-width:400px){...css...}来检测屏幕宽度来判断是否进行其中css的执行。
如果屏幕小，则body内容垂直顺序排列(项目所占空间为其本来大小[flex:auto[1 1 auto]])。

流式布局

每行的项目数固定，会自动分行

HTML

box1      box2
 item1-5      item1-10

CSS

box{
display: flex;
flex-flow: row wrap;
align-content: flex-start;
}
item{
border: solid 1px red;

flex: 0 0 25%;
height: 50px;

box-sizing: border-box;
}

CSS分析
box设定flex-flow(flex-direction + flex-wrap的缩写)为row wrap，即正常水平主轴方向以及可以换行，实现自动换行。
设定多行时的交叉轴对齐方式(align-content)为 flex-start，即顶端对齐。
之后设定item的flex为0 0 25%，即不扩大不缩小，每个占据主轴空间为1/4，但此时的box-size是不包括padding和border的，当item加上border时，就会超出范围，无法达成一行四个的目标。
为了解决这个问题，设定box-size:border-box,即包含padding和border的盒子。此时加上边框和内容构成的一个总盒子占据的主轴空间是25%，达成目标。
