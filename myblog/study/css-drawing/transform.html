transform知识点


transform属性

应用于元素的2D或3D转换。这个属性允许你将元素旋转，缩放，移动，倾斜等。
transform属性的对应值主要分为：
  • rotate-旋转
  • scale-缩放
  • skew-倾斜
  • translate-移动
  • matrix-矩阵
  • div {
    -webkit-transform:rotate(7deg);
    【Safari and Chrome】
    -ms-transform:rotate(7deg);
    【 IE 9 】
    transform:rotate(7deg);
    }

    • perspective-透视 属性

    • .box {
      -webkit-perspective: 100px;
      perspective: 100px;
      }

    transform-origin属性

    要使用这个属性，必须先使用transform属性， transform-origin属性允许改变元素位置。
    2D转换元素可以更改x与y轴、3D还可以改z轴
    默认值为transform-origin: 50% 50% 0;

rotate-旋转

rotate主要值

rotate(angle) 定义 2D 旋转，在参数中规定角度。 rotate3d(x,y,z,angle) 定义 3D 旋转。 rotateX(angle) 定义沿着 X 轴的 3D 旋转。 rotateY(angle) 定义沿着 Y 轴的 3D 旋转。 rotateZ(angle) 定义沿着 Z 轴的 3D 旋转。

rotate-demo

主要HTML

div.card
 div.box+class:rotate/?
  div.fill

对应CSS

.box{
cursor: pointer;
}
.fill{
transition: 0.3s;
}

cursor: pointer;  指示该div上光标变为手型
transition: 0.3s;  指示动画的过度时间为0.3s
transition: property | duration | timing-function | delay;
transition-property:规定设置过度效果的css属性名称。
transition-duration:规定完成过度效果需要多少秒或毫秒。
transition-timing-function:规定速度效果的速度曲线。 【对应值...】
transition-delay:规定过渡效果开始前需要等待的时间，值以秒或毫秒记。

rotate(45deg)

CSS

.rotate:hover .fill {
transform:rotate(45deg);
}

rotate(angle) 定义 2D 旋转，在参数中规定角度。
本处未写带有css属性前缀的属性名，实际运用中需要考虑到兼容性，带上css前缀(-webkit-、-moz-等)

CSS

rotate3d(.5,1,0,45deg);

rotate3d(x,y,z,angle) 定义 3D 旋转。
x,y,z：是一个0到１之间的数值，主要用于描述元素围绕X轴旋转的矢量值。

scale-缩放

scale主要值

scale(x,y) 定义 2D 缩放转换。
scale3d(x,y,z) 定义 3D 缩放转换。
scaleX(x) 通过设置 X 轴的值来定义缩放转换。
scaleY(y) 通过设置 Y 轴的值来定义缩放转换。
scaleZ(z) 通过设置 Z 轴的值来定义 3D 缩放转换。

scale(x,y) 定义 2D 缩放转换。
如果只有一个数值，那么就是xy二者相同。

skew-倾斜

skew主要值

skew(x-angle,y-angle) 定义沿着 X 和 Y 轴的 2D 倾斜转换。
skewX(angle) 定义沿着 X 轴的 2D 倾斜转换。
skewY(angle) 定义沿着 Y 轴的 2D 倾斜转换。

translate-移动

translate主要值

translate(x,y) 定义 2D 转换。
translate3d(x,y,z) 定义 3D 转换。
translateX(x) 定义转换，只是用 X 轴的值。
translateY(y) 定义转换，只是用 Y 轴的值。
translateZ(z) 定义 3D 转换，只是用 Z 轴的值。

matrix-矩阵

matrix主要值

matrix(n,n,n,n,n,n) 定义 2D 转换，使用六个值的矩阵。
matrix3d(n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n) 定义 3D 转换，使用 16 个值的 4x4 矩阵。

perspective-透视

perspective主要值

perspective(n) 为 3D 转换元素定义透视视图。
n是元素距离视图的距离，以像素计，越大离视图越远
