Grid

网格布局(Grid)是最强大的CSS布局方案。
它将网页划分成一个个网格，可以任意组合不同的网格，做出各种各样的布局。
Grid布局与Flex布局有一定的相似性，都可以指定容器内部多个项目的位置。二者之间的区别在于 Flex布局是轴线(行)布局，只能指定项目针对轴线的位置，可看做是一维布局。而Grid布局则是将容器划分成行和列，产生单元格，然后指定项目所在的单元格，可看做是二维布局。

基本概念

容器和项目

采用网格布局的区域，称为容器(container)。容器内部采用网格定位的子元素，称为项目(item)。

注意：项目只能是容器的顶层子元素，不会包含项目的子元素。grid布局只对项目生效。

行和列

水平深色区域为行，垂直深色区域为列。

单元格

行和列的交叉区域，称为单元格(cell)。n行m列会产生n*m个单元格。

网格线

划分网格的线，称为网格线(grid line)。水平网格线划分出行，垂直网格线划分出列。
n行有n+1根水平网格线，m列有m+1根垂直网格线。

容器属性

display属性

display display: grid
display: inline-grid
默认情况下，容器元素都是块级元素，但也可以设为行内元素。

注意：设为网格布局以后，容器子元素(项目)的float、display:inline-block、display: table-cel、 vertical-align和column-*等设置都将失效。

grid-template-areas属性

网格布局允许指定区域(area)，一个区域由单个或多个单元格组成。grid-template-areas属性用于定义区域。
注意：值要用双引号括住。

CSS-1

box{
display: grid;
grid-template-columns: repeat(3,100px)
grid-template-rows: repeat(3,100px)
grid-template-areas: "a b c"
           "d e f"
           "f h i"
}
上面代码先划分出9个单元格，
然后将其定名为a-i的九个区域，
分别对应9个单元格。

多个单元格合并为一个区域写法

grid-template-areas: "a a a"
           "b b b"
           "c c c"
上面代码将9个单元格分成a，b，c三个区域。

布局实例

grid-template-areas: "header header header"
           "main main sidebar"
           "footer footer footer"
上面代码中，
顶部是页眉区域header
底部是页脚区域footer
中间部分则为main和sidebar

如果某些区域不需要利用，
则使用点( . )表示

grid-template-areas: "a . c"
           "b . e"
           "d . f"
上面代码中，中间一列为点，
表示没有用到该单元格
或该单元格不属于任何区域。

注意：区域的命名会影响到网格线。每个区域的起始网格线，会自动命名为 区域名-start ，终止网格线自动命名为 区域名-end 。
如：区域名为header的起始位置水平and垂直网格线叫做header-start。终止位置的水平网格线和垂直网格线叫做header-end。

grid-auto-flow属性

划分网格以后，容器的子元素会按照顺序，自动放置在每一个网格。
默认的放置顺序是“先行后列”，即先填满第一行，再放入第二行。
这个顺序由grid-auto-flow属性决定，默认值是row(先行后列)，另一个值为column(先列后行)

除了设置成row和column，还可以设置成row dense和colunm dense。这两个值用于在某些项目指定位置后，剩下的项目如何自动放置。
1 2 3 4 5 6 7 8 9

CSS

box{
height: 200px;
display: grid;
grid-template-columns: 50px 50px 50px;
grid-template-rows: repeat(auto,1fr);
grid-auto-flow: row;
}
item1,item2{
grid-column-start: 1;
grid-column-end: 3;
}

上例中，3号直接跟着2号排列，1号后是空着的，因为3号默认跟着2号，如改为：
grid-auto-flow: row dense
就如下例，表示先行后列，并且尽可能紧密填满，不出现空格。

justify-items属性
align-items属性
place-items属性

justify-items属性设置单元格内容的水平位置(左中右)， align-items属性设置单元格内容的垂直位置(上中下)。
place-items属性是二者的结合，[place-items: align-items justify-items],如果省略第二个值，那么浏览器认为它与第一个值相等。

  • justify-items: start | end | center | stretch ;
  • align-items: start | end | center | stretch ;
  • start: 对齐单元格的起始边缘。
  • end: 对齐单元格的结束边缘。
  • center: 单元格内部居中。
  • stretch(默认): 拉伸，占满单元格的整个宽度。

  • justify-items: start;
  • align-items: start;

justify-content属性
align-content属性
place-content属性

justify-content属性设置整个内容区域中容器里面的水平位置(左中右)， align-content属性设置整个内容区域中容器里面的垂直位置(上中下)。
place-content属性是二者的结合，[place-content: align-content justify-content],如果省略第二个值，那么浏览器认为它与第一个值相等。

下例以justify-content属性为例，align-content示例只是方向改为垂直。
  • start ———— 对齐容器的起始边框。
  • end ———— 对齐容器的结束边框。
  • center ———— 容器内部居中。
  • stretch ———— 项目大小没有指定时，拉伸占据整个网格容器。
  • space-around ———— 每个项目两侧的间隔相等。所以，项目之间的间隔比项目与容器边框的间隔大一倍。
  • space-between ———— 项目与项目的间隔相等，项目与容器边框之间没有间隔。
  • space-evenly ———— 项目与项目的间隔相等，项目与容器的边框之间也是同样长度的间隔。

grid-auto-columns属性
grid-auto-rows属性

有时，一些项目的指定位置，在现有网格的外部。如网格只有三行，但某项目指定在第五行。此时，浏览器就会自动生成多余的网格来放置项目。
grid-auto-rows/columns用来设置浏览器自动创建的多余网格的行宽列高。它们的写法与 grid-template-rows/columns 完全相同。如不指定，浏览器将完全根据单元格内容的带笑，决定新增网络的行宽和列高。

1 2 3 4 5 6 7 8 9

CSS

box{
display: grid;
grid-template-columns: repeat(3,1fr);
grid-template-rows: repeat(3,1fr);
grid-auto-rows: 20px;
}
item-8{
grid-area: 4/2/5/3;
}
item-9{
grid-area: 5/3/6/4;
}

item 8,9就是在新增的格子里，新增为20px，原本格子行高为50px。

grid-template属性
grid属性

grid-template属性是 grid-template-rows/columns 和 grid-template-areas的合并简写形式。
grid属性是 grid-template-rows/columns 、 grid-template-areas 、 grid-auto-rows/columns 、 grid-auto-flow 这六个属性的合写形式。

项目属性

grid-column-start属性
grid-column-end属性
grid-row-start属性
grid-row-end属性

项目的位置是可以指定的，具体方法就是指定项目的四个边框，分别定位在哪根网络线。

  • grid-column-start属性：左边框所在的垂直网格线。
  • grid-column-end属性：右边框所在的垂直网格线。
  • grid-row-start属性：上边框所在的水平网格线。
  • grid-row-end属性：下边框所在的水平网格线。

CSS

box{
height: 200px;
display: grid;
grid-template-columns: repeat(3,1fr);
grid-template-rows: repeat(auto,1fr);
}
item-1{
grid-column-start: 2;
grid-column-end: 4;
}

box的css，设置display:grid，然后使用grid-template-columns/rows来划分网格，使其成为一个4*3网格。
之后，设置item-1的左边框紧靠第2根垂直网格线，右边框紧靠第4根垂直网格线。[除了使用数字，也可以指定 网格线的名字]，水平网格线采用默认设置。
其余项目没有指定位置，由浏览器自动布局，此时它们的位置由容器的grid-auto-flow属性决定，这个属性的默认值是row，因此会先行后列进行排列，也可改成其余grid-auto-flow值。
